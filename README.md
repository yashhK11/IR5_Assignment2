# People Directory

A React application that fetches and displays random user profiles using the Random User API.

## Live Demo
[View on CodeSandbox](https://codesandbox.io/p/github/yashhK11/IR5_Assignment2)

## Features

- Fetch 3 random user profiles on button click
- Displays user photo, name, age, gender, username and email
- Toggle to show/hide additional info like phone number and full address
- Custom useFetch hook for handling API calls
- Loading and error state handling
